Opis projekta

Navedeni projekt kreiran je od strane studenta redovnog smjera informatike, Alex Mihatović. Navedeni Projekt se sastoji od USE case dijagrama koji se nalazi unutar dokumentacije projekta. Navedeni projekt istodobno objedinjuje python flask, pony orm primjenu baze podataka te kreiranja docker kontejnera. Navedeni elementi su znatno važni za praktičnu primjenu s obzirom da su fundamentalni za kreiranje web aplikacija, zasada ovo je web servis no formiranjem pravilnih funkcionalnosti, povečavanjem apstraktnosti baze podataka te formiranjem detaljnijih i efikasnijih zaahtijeva moguće je formirati web aplikaciju primjerice Web shop. Za takvu primjenu potrebni su stabilni temelji u izradi web servisa te iz tog razloga ovakav projekt je važan.

Winvy stoji za Web shop inventory te baš to je i namjena navedenog web servisa, da upravlja inventarom od web shopa kako bi admin mogao dodavati, uklanjati, uređivati i analizirati artikle koji se nalaze na stanju i u ponudi.
